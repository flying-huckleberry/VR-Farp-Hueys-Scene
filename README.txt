VR FARP HUEYS SCENE

This is a DCS Program mod (not saved games)

Instead of the Su-25T in the hangar, you will see a flight of Hueys
coming into a FARP in staggered left formation.

This is the default scene you will see in the menu as well as the F10 map screen.
